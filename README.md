# 🚀 Práticas de Programação - Lógica & JavaScript

Repositório dedicado ao armazenamento de exercícios, desafios e estudos práticos desenvolvidos durante os meus primeiros passos na programação Web e na linguagem JavaScript. 

O objetivo principal deste projeto foi consolidar os fundamentos da lógica de programação e manipulação de estruturas básicas.

---

## 🛠️ Tecnologias Utilizadas

Uma visão geral das tecnologias e linguagens que compõem os arquivos deste repositório:

* **HTML5:** Estruturação das páginas e interface dos exercícios.
* **JavaScript (ES6):** Aplicação de lógica, estruturas condicionais e laços de repetição.
* **CSS3:** Estilizações básicas para a interface dos mini-projetos.

---

## 📂 O que você vai encontrar por aqui?

Os arquivos estão divididos entre conceitos fundamentais e pequenos jogos práticos:

### 🎮 Mini-Projetos / Jogos
* `JogoDaVelha.html`: Implementação clássica do jogo da velha em JavaScript.
* `Tabuada.html`: Gerador automático de tabuadas dinâmicas.

### 🧠 Estruturas de Controle e Repetição
* `condicoes.html`: Práticas utilizando estruturas condicionais (`if`, `else if`, `else`).
* `switchcase.html`: Menu de escolhas estruturado com `switch/case`.
* `while.html` e `for.html`: Demonstrações e repetições usando laços de iteração clássicos.
* `for-in.html`: Iteração específica sobre propriedades de objetos/arrays.

### 📊 Estruturas de Dados (Coleções)
* `array.html` e `arrayMultidimensional.html`: Manipulação, criação e iteração de vetores e matrizes de dados.
* `ArrayNotas.html`: Exercício prático de manipulação de notas acadêmicas armazenadas em arrays.

### ⚡ Conceitos Avançados de Lógica
* `funcoes.html`: Criação, passagem de parâmetros e retorno de funções isoladas.
* `recursividade.html`: Estudos práticos sobre funções que chamam a si mesmas para resolver problemas.

---

## 🚀 Como visualizar os exercícios

Como todos os arquivos foram criados utilizando tecnologias puras do ecossistema Web (`HTML`, `JS` e `CSS`), você não precisa instalar nenhuma dependência:

1. Faça o clone deste repositório:
   ```bash
   git clone https://github.com
   ```
2. Navegue até a pasta do projeto.
3. Abra qualquer um dos arquivos `.html` diretamente no seu navegador de preferência para ver o código em execução.
