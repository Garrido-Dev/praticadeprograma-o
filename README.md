Estrutura de Arrays 

push = adiciona elementos
unshift = coloca elemento em 1º
pop = retira o ultima elemento
shift = retira primeiro elemento
splice = estrutura complexa (lugar, quantidade, adiciona item) ---> lugar e quantidade para remover.
sort = ordena os itens alfabeticos e numéricos.
